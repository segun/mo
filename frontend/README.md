### BUILDING/RUNNING

1. Clone this repo
2. Install dependencies
    `npm install`
3. You can run using
    `npm run start`
4. Or you can build a deployable app using
    `npm run build`
5. App will be built into `/app` folder that can deployed to a webserver (nginx, apache, etc)
